# TODO APP with EXPRESS + REACT + REDUX.

## Stack

- Node.js
- Express.js
- React.js
- Redux.js
- Socket.io
- MongoDB

## Commands

- build prod  
`build:prod`  
- build dev  
`build dev`  
- launch linter  
`npm lint`  
- launch tests  
`npm test`  
- check tests coverage  
`npm coverage`  
- analyse bundle stats  
`bundle:stats && bundle:analyse`
